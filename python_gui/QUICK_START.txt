🎮 HOW TO USE

AUTHENTICATION:
→ Click "Sign In / Sign Up" in header
→ Create account or use social login
→ Set up profile with username/photo

PET ADOPTION:
→ Go to Adoption page
→ Search or filter by species
→ Click pet card → View details → ADOPT
→ Complete multi-step form

SERVICE BOOKING:
→ Go to Services page
→ Click "Book Now"
→ Select date & time on calendar
→ Fill pet details → Review → Confirm

SHOPPING:
→ Browse Products
→ Add to Cart
→ Adjust quantities
→ Checkout

WISHLIST:
→ Click ♥ on any pet/service card
→ View all in Wishlist page
